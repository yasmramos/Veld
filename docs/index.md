# Veld Framework Documentation

Welcome to the Veld Framework documentation. Veld is a high-performance dependency injection and AOP framework for Java.

## Modules

- [veld-annotations](./modules/annotations.md) - Core annotations for Veld
- [veld-aop](./modules/aop.md) - Aspect-Oriented Programming support
- [veld-processor](./modules/processor.md) - Annotation processor for compile-time code generation
- [veld-weaver](./modules/weaver.md) - Bytecode weaving capabilities
- [veld-maven-plugin](./modules/maven-plugin.md) - Maven plugin for Veld
- [veld-security](./modules/security.md) - Security features
- [veld-spi](./modules/spi.md) - Service Provider Interface
