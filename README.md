# Veld

A Java Maven project.

## Requirements

- Java 17 or higher
- Maven 3.6+

## Build

```bash
mvn clean install
```

## Run

```bash
java -jar target/veld-1.0.0-SNAPSHOT.jar
```

## Test

```bash
mvn test
```
